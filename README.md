# 📚 Product Page – Book Pre-Order Website

A responsive product page built with HTML, CSS, and vanilla JavaScript, featuring an interactive image gallery, lightbox modal, and dynamic cart functionality.

🔗 Live Site: https://amycarolparks318.github.io/

---

## ✨ Features

- 🖼️ Interactive image gallery with thumbnail selection
- 🔍 Lightbox modal with next/previous navigation
- 🛒 Dynamic shopping cart with quantity updates
- 📱 Fully responsive design for mobile and desktop
- 🎯 Clean UI with modern layout and styling
- ❌ Click-outside-to-close cart functionality

---

## 🧠 What I Learned

- Managing DOM interactions and event listeners in vanilla JavaScript
- Handling state (cart, quantity, image index) without frameworks
- Building reusable UI patterns like modals and popups
- Debugging cross-page JavaScript issues
- Writing cleaner, more maintainable CSS

---

## 🛠️ Technologies Used

- JavaScript (ES6)
